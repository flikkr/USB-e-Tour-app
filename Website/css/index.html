A Quick Guide On Room Finder

Display

The top half of the screen is for users to input the information for finding a route, and the lower half displays the floor plan.

Input Information

There are instructions on what to input within the text box, so Current Place is the current room that you are in, and it requires you to enter a room code (which you can find on the floor plans). Therefore Destination requires you to enter a room code of the room you want to find.

Preferred Access

Toggle between using stairs or lifts to get to your room, but only if the start and destination are on different floors.

Search

Finally you can now search for a room by pressing the Search button, or to clear the fields and start again, simply press the Clear button.
